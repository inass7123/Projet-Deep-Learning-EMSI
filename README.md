# Projet de Deep Learning - EMSI 2026

Projet de fin de module portant sur la conception, l'implémentation et l'analyse critique de modèles de Deep Learning sous PyTorch.

## Contenu du projet
- `Projet_DeepLearning_INASS_MAZOUZI.ipynb` : Notebook complet incluant :
  - **MLP** pour données tabulaires (Breast Cancer Wisconsin).
  - **CNN** pour la vision par ordinateur (Fashion-MNIST).
  - **Seq2Seq (GRU)** pour le traitement de séquences (Tatoeba).
- [ Rapport du projet](./Rapport_Deep learning_Inass_MAZOUZI.pdf)
## Pipeline de traitement
Le projet suit une démarche d'ingénierie rigoureuse pour démontrer le biais inductif :
1. **Préparation des données** : Normalisation, gestion du déséquilibre et prévention du Data Leakage.
2. **Architecture** : Adaptation structurelle selon la nature des données.
3. **Entraînement** : Utilisation de l'optimiseur Adam, gestion du Dropout et monitoring des courbes de perte.
4. **Évaluation** : Analyse via matrices de confusion et interprétabilité.
